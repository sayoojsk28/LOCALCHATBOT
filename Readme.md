# 🤖 My AI Chatbot (Qwen2.5 + Streamlit)

A lightweight AI chatbot built using **Hugging Face Transformers** and **Streamlit**, powered by the local model **Qwen2.5-0.5B-Instruct**.

The app provides a simple chat interface where users can interact with a locally running LLM.

---

## 🚀 Features

- Local LLM inference (no API required)
- Chat-style UI using Streamlit
- Conversation memory (session-based)
- Fast lightweight model (Qwen2.5-0.5B-Instruct)
- Clean and minimal interface

---

## 🧠 Model Used

- Qwen/Qwen2.5-0.5B-Instruct  
A small instruction-tuned model from Hugging Face optimized for lightweight chat applications.
